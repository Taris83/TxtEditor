# TxtEditor

Une implémentation d'un bloc-note like allégé en C#.

## Classe: Editor
Classe contenant toutes les méthodes basic d'un éditeur basé sur le composant RichTextBox (Winforms).

## Fonctions publiques
- Nouveau fichier
- Ouvrir fichier
- Sauvegarder / Sauvegarder Sous
- Annuler / Retablir
- Couper / Copier / Coller
- Tout sélectionner
